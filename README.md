# The RayTracer Challenge

This is an Scala implementation of the fantastic The Ray Tracer Challenge, check it out [here!](http://raytracerchallenge.com/)

### Build, test and running

```sh
$ ./mill raytracer.compile
```

```sh
$ ./mill raytracer.test
```

```sh
$ ./mill raytracer.run
```

### Dependencies

  - mill
